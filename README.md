# OpenClaw + Opencode Bridge

[![English](https://img.shields.io/badge/Language-English-blue)](README.en.md)
[![中文](https://img.shields.io/badge/语言-中文-red)](README.zh-CN.md)

OpenAI-compatible bridge for OpenClaw to use local opencode (including free-AI path).

Default package layout:
- `ghcr.io/alphabaijinde/openclaw-opencode-bridge:latest` = all-in-one image (`openclaw + opencode + bridge`)
- `ghcr.io/alphabaijinde/openclaw-opencode-bridge:bridge-only` = bridge-only sidecar tag for the three-container stack
